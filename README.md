# HelloMod

Mod pro Terraria vytvořený pomocí frameworku **tModLoader**.

Přidává:
- **Sluneční Meč** – mečíř, který při švihnutí vystřelí ohnivou kouli a zapaluje nepřátele
- **Ohnivá Koule** – magický projektil s efektem průletu a výbuchu při dopadu
- **Přátelský Cestovatel** – přátelské NPC spawující se na povrchu za dne s náhodným dialogem

## Jak sestavit mod

1. Spusť Terraria s nainstalovaným tModLoader
2. **Workshop → Develop Mods → HelloMod → Build + Reload**

Sestavený soubor `.tmod` se uloží do složky `Mods/`.
